Markdown Basics

Format the text in your R Markdown file with Pandoc’s Markdown, a set of markup annotations for plain text files. When you render your file, Pandoc transforms the marked up text into formatted text in your final file format:


You can use Pandoc’s Markdown to make:

  • Headers
  • Lists
  • Links
  • Images
  • Block quotes
  • Latex equations
  • Horizontal rules
  • Tables
  • Footnotes
  • Bibliographies and Citations
  • Slide breaks
  • Italicized text
  • Bold text
  • Superscripts
  • Subscripts
  • Strikethrough text

Read more about markdown at Pandoc’s Markdown or consult the quick reference in the R Markdown Cheatsheet.
