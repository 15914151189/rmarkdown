Presentations

You can generate a slide presentation from the same .Rmd file that you use to render reports. Each of R Markdown’s presentation formats will intuitively divide your content into slides, with a new slide beginning at each first or second level header.

Insert a horizontal rule (***) into your document to create a manual slide break.

You can learn more about each of R Markdown’s slide formats here:

  • ioslides slideshows
  • reveal.js slideshows
  • slidy slideshows
  • Beamer Slideshows

Exercise

Add a title and author to the YAML of your your document and remove the table of contents. Then change the output field to ioslides_presentation and add slide breaks to make your report resemble the slide deck below.

R Markdown combines the core syntax of **markdown** (an easy to write plain text format) with embedded **R** code chunks that are run so their output can be included in the final document.
